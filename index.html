Base Set Charizard - 1999

Type:

Charizard is een Fire-type Pokémon en staat bekend om zijn krachtige vuuraanvallen.

Uitgave:

De Base Set Charizard werd voor het eerst uitgebracht in 1999 in Noord-Amerika als onderdeel van de allereerste Pokémon TCG-set, bekend als de "Base Set." In Japan werd deze kaart voor het eerst uitgebracht als "Charizard"

Eigenschappen:

De Base Set Charizard wordt vaak beschouwd als een van de meest iconische en gewilde kaarten in de TCG-geschiedenis. Deze kaart heeft een indrukwekkende HP-waarde en krachtige aanvallen, waaronder "Fire Spin."

Expedition Charizard - 2002

Type:

Charizard LV.X-kaarten behoren tot een speciale categorie van Pokémon-kaarten die bekend staan als “Level-Up” kaarten.

Uitgave:

De Base Set Charizard werd voor het eerst uitgebracht in 1999 in Noord-Amerika als onderdeel van de allereerste Pokémon TCG-set, bekend als de "Base Set." In Japan werd deze kaart voor het eerst uitgebracht als "Charizard"

Eigenschappen:

Charizard is een van de meest iconische Pokémon en staat bekend om zijn indrukwekkende aanvallen en ontwerp. Deze kaart is een omgekeerde holografische versie, wat betekent dat de glans zich concentreert op de afbeelding van Charizard en de tekst, wat een uniek visueel effect geeft.

Charizard LV.X - 2007

Type:

Charizard LV.X-kaarten behoren tot een speciale categorie van Pokémon-kaarten die bekend staan als “Level-Up” kaarten.

Uitgave:

Deze Charizard LV.X-kaart is voor het eerst uitgebracht in de "Diamond and Pearl" set van de Pokémon Trading Card Game. Deze set werd uitgebracht in 2007 en introduceerde nieuwe Pokémon en kaarten die gebaseerd waren op de Diamond en Pearl videogames.

Eigenschappen:

Charizard LV.X-kaarten vertegenwoordigen geëvolueerde versies van Charizard met verbeterde aanvallen en vaardigheden. Ze hebben meestal hogere HP (Hit Points) en krachtigere aanvallen dan standaard Charizard-kaarten.

Charizard-EX - 2014

Type:

Charizard-EX behoort tot de categorie EX-kaarten, die krachtige en zeldzame Pokémon-kaarten zijn in de TCG.

Uitgave:

Charizard-EX is voor het eerst uitgebracht in de "XY Flashfire" set van de Pokémon Trading Card Game. Deze set werd uitgebracht in 2014.

Eigenschappen:

Charizard-EX staat bekend om zijn hoge aanvalskracht en HP (Hit Points). Deze kaart heeft vaak krachtige aanvallen en vaardigheden die het tot een geduchte tegenstander maken in gevechten.

Charizard-GX - 2017

Type:

Charizard-GX behoort tot de GX-kaarten, een speciale categorie van krachtige Pokémon-kaarten in de TCG.

Uitgave:

Charizard-GX is voor het eerst uitgebracht in de "Burning Shadows" set van de Pokémon Trading Card Game. Deze set werd uitgebracht in 2017.

Eigenschappen:

Charizard-GX, de belichaming van vuurkracht, schittert op deze kaart met imposante kenmerken. Met een solide HP kan hij de tand des tijds doorstaan, terwijl zijn aanvallen zoals "Flare Blitz" vurige vernietiging brengen. De GX-aanval, een unieke troef, voegt een element van verrassing toe aan het spel.

Charizard VSTAR - 2021

Type:

Charizard-GX behoort tot de GX-kaarten, een speciale categorie van krachtige Pokémon-kaarten in de TCG.

Uitgave:

Charizard VSTAR werd voor het eerst uitgebracht in de "Evolving Skies" set van de Pokémon Trading Card Game. Deze set werd uitgebracht in 2021.

Eigenschappen:

Charizard VSTAR is een extreem krachtige kaart met hoge aanvalswaarden en speciale vaardigheden die het een geduchte tegenstander maken in gevechten.
