Base Set Charizard - 1999

Type: Charizard LV.X-kaarten behoren tot een speciale categorie van Pokémon-kaarten die bekend staan als “Level-Up” kaarten.

Uitgave: De Base Set Charizard werd voor het eerst uitgebracht in 1999 in Noord-Amerika als onderdeel van de allereerste Pokémon TCG-set, bekend als de "Base Set." In Japan werd deze kaart voor het eerst uitgebracht als "Charizard"

Uitgave: Deze Charizard LV.X-kaart is voor het eerst uitgebracht in de “Diamond and Pearl” set van de Pokémon Trading Card Game. Deze set werd uitgebracht in 2007 en introduceerde nieuwe Pokémon en kaarten die gebaseerd waren op de Diamond en Pearl videogames.
